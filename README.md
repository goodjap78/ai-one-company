# AI ONE COMPANY

AI 기반 서비스를 빠르게 기획, 개발, 출시하기 위한 제품/엔지니어링 저장소입니다.

## First Product
- `apps/todays-menu`: AI 메뉴 추천 앱, 오늘의 메뉴

## Repository Structure
```text
apps/          제품 앱
packages/      공통 UI, shared logic, config
backend/       API 서버, DB, 인증
ai/            AI 추천 로직, 프롬프트, 평가
docs/          제품/기술/사업 문서
design/        디자인 토큰, 컴포넌트 기준
assets/        이미지, 아이콘, 브랜드 자산
scripts/       자동화 스크립트
.github/       GitHub Actions, Issue/PR 템플릿
```

문서가 먼저이고, 코드는 문서를 따릅니다.
